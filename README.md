## How to use

### Installation
```bash
git clone https://github.com/yuru-yuri/Manga-Downloader.git
cd Manga-Downloader
# install requirements
pip -r requirements.txt
```
### Downloading
```bash
./manga.py -i -p -u http://manga-url-here/manga-name -d /manga/destination/path/
# or interactive mode
./manga.py -i -p
```
### Help
```bash
./manga.py -h
# or
./manga.py --help
```
